# 涂装车间能耗预测系统

一个基于 Web 的涂装车间能源消耗预测系统，支持多车间、多设备的能耗计算和成本分析。

## 🌟 功能特点

- **双车间支持** - 支持一车间和二车间独立配置
- **可视化流程图** - 直观的工艺流程展示，色彩分区
- **模块化设备选择** - 灵活选择需要的生产设备
- **精确时间计算** - 分别计算升温、生产、降温时间
- **环境温度因子** - 根据环境温度自动调整能耗预测
- **实时成本计算** - 自动计算工业水、天然气、电能的用量和费用
- **数据可编辑** - 支持修改设备基础能耗数据并保存

2. 打开 `coating-energy-predictor.html` 文件即可使用

## 🛠️ 文件说明

- `coating-energy-predictor.html` - 主预测页面
- `equipment-data.html` - 设备基础能耗数据管理页面
- `workshop-data.js` - 车间设备配置数据文件

## 💡 使用方法

1. 打开主页面后，选择车间（一车间或二车间）
2. 在流程图或列表中勾选需要使用的设备
3. 设置每个设备的生产时间
4. 输入环境温度、日产量、能源单价等参数
5. 点击"计算能耗预测"查看结果

### 管理员功能

点击"设备基础能耗"按钮可以编辑每个设备的基础能耗数据：
- 升温时间
- 降温时间
- 工业水消耗（吨/小时）
- 天然气消耗（m³/小时）
- 电能消耗（kWh/小时）

修改后的数据会保存在浏览器本地存储中。

## 🎨 工艺流程

系统包含以下工段：

- **前处理工段** - 白车体擦净、热水洗、预脱脂、脱脂、锆化等
- **电泳工段** - 电泳槽、电泳烘房、UBS喷涂等
- **密封胶工段** - PVC喷涂、密封胶烘房等
- **面漆工段** - 面漆喷涂、面漆烘房等
- **辅助设备** - 废气处理系统、循环水系统等
